# ISC2 Certified in Cybersecurity CC Study Guide and Notes

A clean domain-by-domain study guide for the ISC2 Certified in Cybersecurity (CC) certification. These notes are written for learning, revision, and exam preparation without needing to move through many folders or scattered files.

The goal of this repository is to make CC exam topics easier to study by keeping each domain in one readable file. Each domain includes study sections, key concepts, important terms, practice tasks, quick-check questions, and a final review checklist.

## How to Use These Notes

1. Start with Domain 1 and study each domain in order.
2. Read the explanations before memorizing terms.
3. Review the important terms until you can explain them in your own words.
4. Complete the practice tasks to connect concepts with real security work.
5. Use the quick-check questions to test recall.
6. Use the final review checklist before moving to the next domain.
